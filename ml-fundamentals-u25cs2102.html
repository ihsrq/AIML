Machine Learning Fundamentals

Introduction to machine learning concepts, supervised and unsupervised learning, and practical implementation using Python.

Back to Semester

Weekly Learning Goals

  • Understand ML fundamentals: types of learning, bias-variance tradeoff, and model evaluation.
  • Master supervised learning: linear regression, logistic regression, and decision trees.
  • Explore unsupervised learning: clustering, dimensionality reduction, and association rules.
  • Implement ML algorithms using Python libraries (scikit-learn, pandas, numpy).
  • Learn feature engineering, model selection, and hyperparameter tuning.
